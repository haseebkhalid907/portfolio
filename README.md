# Haseeb Khalid — Portfolio

A modern, professional portfolio website built with **Next.js 15**, **TypeScript**, **Tailwind CSS**, and **Framer Motion**.

## Features

- Dark theme with gradient accents and glass morphism effects
- Smooth scroll animations with Framer Motion
- Responsive design (mobile-first)
- Interactive project showcase with featured/all toggle
- Categorized skills display
- Contact form with mailto integration
- Static export for deployment anywhere
- SEO optimized with Open Graph meta tags

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion
- **Icons:** Lucide React

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# The static output will be in the `out/` directory
```

## Deployment

### Vercel (Recommended)

1. Push to GitHub
2. Import project on [vercel.com](https://vercel.com)
3. Deploy automatically

### Netlify

1. Push to GitHub
2. Import project on [netlify.com](https://netlify.com)
3. Set build command: `npm run build`
4. Set publish directory: `out`

### GitHub Pages

1. Push to GitHub
2. Enable GitHub Pages in repo settings
3. Use the `out/` directory or set up a GitHub Action
